# Gamemode Ideen:

## Räuber und Gendarm:

* Wenn ein Gendarm den Räuber berührt wird dieser ins Gefängnis teleportiert.
* Das Gefängnis kann von den Räubern über ein P-Schalter geöffnet bzw. despawn werden, damit die gefangenen fliehen können.

### Features

* eventuell costum oder schon in Game Outfides für Räuber und Gendarmen die sich automatisch wechseln 


## 15 Zählt 

* Alle verstecken sich 
* Der fänger darf sich dann nur wenig bewegen. (Per Dive)
* Nachdem der Sucher die Dives gemacht hat und einen Knopf drückt wird sein Bildschirm schwartz oder wird in snapshot mode gebracht.
* Dann haben die Verstecker eine bestimmte Zeit den Sucher zu berüheren und müssen sich dann wieder verstecken.
* Nach der Zeit wird der Sucher wieder aus dem Snapshot modus oder dem Schwartzen Bild rausgeholt
* Die Zeit wird nach jedem weglaufen weniger

### Features
* Man kann die Zeit induviduel anpassen im Menü oder im Server.
